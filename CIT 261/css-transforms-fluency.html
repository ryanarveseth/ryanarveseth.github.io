CSS Transforms
  • Transforms allow you to move, rotate, skew, and scale elements.
    I typically use transforms for buttons to make them more dynamic and responsive.
  • Transform Methods
    • translate(x, y)
      • Moves the element from the current position (x, y) pixels
    • rotate(#deg)
      • Rotates the element by x degrees
    • scaleX(#)
      • Increases the horizontal scale of the element by #
    • scaleY(#)
      • Increases the vertical scale of the element by #
    • scale(#)
      • Increases the entire scale of the element by #
    • skewX(#deg)
      • SkewX skews an element along the x axis by # degrees
    • skewY(#deg)
      • SkewY skews an element along the y axis by # degrees
    • skew(#degX, #degY)
      • Skews an entire element by # degrees (x and y)
    • matrix(scaleX, skewY, skewX, scaleY, translateX, translateY)
      • Combines all of the above methods into one (overkill!)
