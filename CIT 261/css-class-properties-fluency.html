CSS Class Properties
Class Query
  • first, getElementsByClassName
    • [].slice.call() --- call each of the elements
    • forEach(function (variable) --- run this function for each
    • Then you can choose the class property (properties) you want to change
Modifying Classes
  • className
    • Gets (or sets) the class value
  • element.classList.add()
    • Add to the class values
  • element.classList.toggle()
    • Toggles the class on or off!
    • ex. classList.toggle('active')
  • element.classList.contains()
    • Check whether or not a class value exists
  • element.classList.replace()
    • Replace a class value with a new one!
  • element.classList.remove()
    • Remove a class value from an element
CSS Class Manipulation
In the Game Modifiers (settings), you can toggle the background between black and clear. This uses the classList.toggle() method

    • Any time 2 balls collide, you get points!
    • If a blue ball hits an orange ball, the orange ball turns blue!
    • If a blue ball hits another blue ball, one 'dies'
