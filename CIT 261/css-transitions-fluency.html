CSS Transitions
  • Transitions make animations or changes smoothe instead of rigid. An element:hover property can be transitioned slowly, instead of jumping from one state to the other.
  • Properties
    • transition:
    • transition-delay:
    • transition-duration:
    • transition-property:
    • transition-timing-function:
      • ease - slow start, fast middle, slow end (slower than start)
      • linear - same speed from start to finish
      • ease-in - slow start, faster middle and end
      • ease-out - fast start and middle, slow end (same as start)
      • ease-in-out - slow start and end
