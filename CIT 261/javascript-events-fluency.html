JavaScript Events

Common JavaScript Events
  • onchange
    • An element has been changed
  • onclick
    • The user clicked on an element
  • onmouseover
    • The user moved the mouse over an element
  • onmouseout
    • The user moved the mouse out of the element
  • onkeydown
    • The user pushes a key on the keyboard down
  • onload
    • The browser has loaded the element
