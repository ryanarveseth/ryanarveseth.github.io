Objects
  • Properties to objects are attributes
    • properties can be anything, like color, mass, radius, etc.
    • The example below has several properties: x, y, vx, vy, alive, radius, mass, started, and isBlue
  • Object properties are accompanied by property values
    • Object.property = value;
      • OR
    • Object["propertyName"] = value;
    • In the example below, each property is defined with 'this.propertyName'
  • Methods are functions within the object that accomplish a task.
    • Object.method();
    • In the example below, you can see the kill() & draw() methods.

BOUNCE (Simple Game)
  • Balls are objects
  • The crosshair is an object as well
  • Rules
    • Get the largest number of points in 30 seconds!
    • Any time 2 balls collide, you get points!
    • If a blue ball hits an orange ball, the orange ball turns blue!
    • If a blue ball hits another blue ball, one 'dies'
  • (Different ball collisions make different noises)
