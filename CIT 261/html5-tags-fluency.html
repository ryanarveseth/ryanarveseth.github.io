HTML5 Media Tags
  • Canvas
    • The canvas tag is used to draw graphics on the fly, using scripting. Canvas is only a container for graphics.
  • Video
    • Use this tag to show a video in HTML. You can add video controls too.
  • Audio
    • Use this tag to play audio in HTML. You can add audio controls too. You can embed and hide audio tags in the html doc and access them in a game (spoiler for below).
