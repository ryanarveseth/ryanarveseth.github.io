DOM Manipulation

DOM Manipulation
  • createElement()
    • Creates an Element node with the specified name
  • appendChild()
    • Adds a new node to the end of the parent node's list of children nodes
  • insertBefore()
    • Adds a new node to the beginning of the parent node's list of children nodes
  • removeChild()
    • Removes a specified child node of the specified element.
  • replaceChild()
    • Replace a specified node of the specified element.
