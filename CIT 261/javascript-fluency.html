JavaScript Loops
  • Types of JavaScript Loops
    • for
      • Loops for a set number of times until it reaches the 'go until' point.
      • Syntax: for (start; go until; increment by)
      • Example: for (i = 0; i < 10; i++)
    • for/in
      • Loops over each item in an object.
      • Syntax: for (variable in object)
      • Example: for (i in names)
    • for/of
      • Loops over iterable data structures like Strings, Maps, NodeLists, Arrays
      • Syntax: for (variable of iterable item)
      • Example: for (i in names)
    • while
      • Loops while a condition evaluates to true (the condition is checked BEFORE the loop begins)
      • Syntax: while (condition)
      • Example: while (amAwesome)
    • do/while
      • Runs the code, checks the condition status, then loops until the condition evaluates to false
        (the condition is checked AFTER the loop begins)
      • Syntax: do {...} while (condition);
      • Example: do {...} while (amAwesome);

JavaScript Conditions
  • Types of JavaScript Conditions
    • if
      • Checks a condition and runs code IF the condition evaluates to true.
      • Syntax: if (condition) { }
      • Example: if (isHungry) { eat(); }
    • else
      • Goes after an if statement and runs code if the condition evaluates to false
      • Syntax: else { }
      • Example: else { dontEat(); }
    • else if
      • Goes after an if statement and runs code if the new condition evaluates to true
        (and the previous condition evaluates to false)
      • Syntax: else if (condition) { }
      • Example: else if (isHangry) { beHangry(); }
    • switch
      • Syntax: switch (expression) {
        case (condition1):
        case (condition2):
        case (condition3):
        }
      • Example:
        switch (feelings) {
        case (happy):
        case (sad):
        case (hungry):
        case (angry):
        }
Example of a Condition

JavaScript Functions & Parameters
  • Functions
    • Example:
      function myFunction() { }
      OR
      function myFunction(parameters...) { }
    • A block of code which is designed to perform a specific task
    • A function is executed when something calls it.
    • You can pass parameters to functions when you want it to do something
      with the value
  • Parameters
    • Example: function myFunction(parameter1, parameter2, parameter3) { }
    • Parameters are passed through to functions, as seen in the fib(n) function to the right.
    • In that case, the function takes n, and uses it to accomplish its task
Example of a function
// Let's do the fibonacci sequence
// fib takes in a number and...
// returns the nth number in fib sequence
function fib(n) {
// first 2 numbers in sequence are 0 & 1
arr = [0, 1];
for (let i = 2; i <= n; i++) {
// add next num to the array
arr.push(arr[i-1] + arr[i-2]);
}
// return result to whoever called the function
return arr[n];
}

JavaScript Variables
  • Variables
    • var
      • Redeclaring variables with var can make problems
      • When you change a var inside, or outside of a block - { } - it changes all of the other instances of it
      • Try the example to the right - it will change a var inside of a block.
        Outside the block, it was also changed.
      • Does not have block scope
    • let
      • Redeclaring variables with let can solve the problems stated above.
      • Redeclaring a variable inside a block will not change it outside the block.
      • Has block scope - (cannot be accessed from outside of block)

JavaScript Arrays & Associative Arrays
  • Arrays
    • var arr = [item1, item2, item3, etc...];
    • Can be accessed with indexes -> arr[index]
    • Array indexes start at 0. arr[0] is the 1st item in the array!
  • Associative Arrays
    • arr["make"] = "Toyota";
    • arr["model"] = "Camry";
    • Associative arrays are turned into Objects by JavaScript
    • Since associative arrays are redefined into objects, some of the
      normal array methods won't produce their normal results!
Example of Arrays
