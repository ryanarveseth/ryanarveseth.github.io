JSON.stringify(object)
  • Receives Data as an object and changes it into a JSON formatted string.
  • Date objects are not allowed. JSON.stringify() will convert dates to strings.
  • Functions are also not allowed as objects - so 'number: function () { return 5; }' won't work.
    • (Unless you convert the function to a string first!)
