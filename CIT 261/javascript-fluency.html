JavaScript Loops
  • Types of JavaScript Loops
    • for
      • Loops for a set number of times until it reaches the 'go until' point.
      • Syntax: for (start; go until; increment by)
      • Example: for (i = 0; i < 10; i++)
    • for/in
      • Loops over each item in an object.
      • Syntax: for (variable in object)
      • Example: for (i in names)
    • for/of
      • Loops over iterable data structures like Strings, Maps, NodeLists, Arrays
      • Syntax: for (variable of iterable item)
      • Example: for (i in names)
    • while
      • Loops while a condition evaluates to true (the condition is checked BEFORE the loop begins)
      • Syntax: while (condition)
      • Example: while (amAwesome)
    • do/while
      • Runs the code, checks the condition status, then loops until the condition evaluates to false
        (the condition is checked AFTER the loop begins)
      • Syntax: do {...} while (condition);
      • Example: do {...} while (amAwesome);

JavaScript Conditions
  • Types of JavaScript Conditions
    • if
      • Checks a condition and runs code IF the condition evaluates to true.
      • Syntax: if (condition) { }
      • Example: if (isHungry) { eat(); }
    • else
      • Goes after an if statement and runs code if the condition evaluates to false
      • Syntax: else { }
      • Example: else { dontEat(); }
    • else if
      • Goes after an if statement and runs code if the new condition evaluates to true
        (and the previous condition evaluates to false)
      • Syntax: else if (condition) { }
      • Example: else if (isHangry) { beHangry(); }
    • switch
      • Syntax: switch (expression) {
        case (condition1):
        case (condition2):
        case (condition3):
        }
      • Example:
        switch (feelings) {
        case (happy):
        case (sad):
        case (hungry):
        case (angry):
        }
